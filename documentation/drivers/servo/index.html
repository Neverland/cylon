Servo

Driver

Provides an interface for making servos easy to work and interact with from Artoo.

How to establish a connection and setup the driver

For Arduino:

Cylon = require('cylon')

Cylon.robot
  connection:
    name: 'arduino', adaptor: 'firmata', port: '/dev/ttyACM0'

  device:
    name: 'servo', driver: 'servo', pin: 3

Commands:

angle

Moves the servo to the specified angle, angle must be an integer value between 0 and 180.

currentAngle

Returns the current angle of the servo, an integer value between 0 and 180.

Events

  • start

    Sent when the device has been started and is ready to use.

How to use

Example moves a servo from angle 0 to 90 to 180.

Cylon = require('..')

Cylon.robot
  connection:
    name: 'arduino', adaptor: 'firmata', port: '/dev/ttyACM0'

  device:
    name: 'servo', driver: 'servo', pin: 3

  work: (my) ->
    angle = 0
    increment = 90
    every 1.seconds(), ->
      angle += increment
      my.servo.angle(angle)
      console.log("Current Angle => #{ my.servo.currentAngle() }")
      increment = -increment if (angle is 0) or (angle is 180)

.start()
