Pebble

Driver

Allows user to interact with Artoo devices using a Pebble smart watch.

How to establish a connection and setup the driver

require 'artoo'
connection :pebble, :adaptor => :pebble, :port => "/dev/rfcomm0", :id => "378B"
device :watch, :driver => :pebble

How to use

Example receiving button events from a Pebble smart watch.

require 'artoo'

connection :pebble, :adaptor => :pebble, :port => "/dev/rfcomm0", :id => "378B"
device :watch, :driver => :pebble

work do
  on watch, :media_control => :button_push
end

def button_push(caller, data)
  puts data.button unless data.nil?
end
