Motor

Driver

Provides motor manipulation tools for artoo like on/off, speed modulation using PWM, use of bidirectional motors using forward/backward pins.

How to establish a connection and setup the driver

For Arduino:

Cylon = require('cylon')

Cylon.robot
  connection:
    name: 'arduino', adaptor: 'firmata', port: '/dev/ttyACM0'

  device:
    name: 'motor', driver: 'motor', pin: 3

Commands:

turnOn

Starts the motor.

turnOff

Stops the motor.

toggle

Sets the state of the motor to the oposite of the current state, if motor is on then sets it to off.

speed(speed_value)

ets the speed of the motor to the value provided in the speed param, speed value must be an integer between 0 and 255.

currentSpeed

Returns the current speed of the motor as an integer between 0 and 255.

Events

  • start

    Sent when the device has been started and is ready to use.

How to use

Example of a dc motor with speed changes using PWM.

  Cylon = require('..')

  Cylon.robot
    connection:
      name: 'arduino', adaptor: 'firmata', port: '/dev/ttyACM0'

    device:
      name: 'motor', driver: 'motor', pin: 3

    work: (my) ->
      speed = 0
      increment = 5
      every 0.05.seconds(), ->
        speed += increment
        my.motor.speed(speed)
        console.log("current speed => #{ my.motor.currentSpeed() }")
        increment = -increment if (speed is 0) or (speed is 255)

  .start()
